# Instructions for AI Agents

When working on this project, follow this onboarding sequence:

## 1. Read the README
Start by reading [README.md](./README.md) to understand:
- Project overview and purpose
- Documentation structure
- Architecture overview
- Current project status

## 2. Read the Current Task
Read [CURRENT_TASK.md](./CURRENT_TASK.md) to understand:
- What we're working on right now
- Current opportunity/problem being solved
- Implementation context
- Next steps

## 3. Context Hierarchy
- **CURRENT_TASK.md** - Immediate work context
- **PLAN.md** - Now/Next/Later roadmap
- **docs/opportunities/** - Detailed problem/solution exploration
- **PRODUCT_BRIEF.md** - Complete product vision
- **docs/adr/** - Architecture decisions

Always start with README → CURRENT_TASK before diving into implementation.
